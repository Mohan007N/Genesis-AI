## 📖 Overview

Genesis AI is an AI-powered software reverse engineering platform that understands application screenshots and reconstructs the entire software architecture.

Unlike traditional screenshot-to-code tools, Genesis AI doesn't stop at generating HTML or React components. It intelligently predicts application workflows, business logic, backend APIs, database schemas, security considerations, and deployment-ready project structures.

The goal is to reduce weeks of software development into minutes.

## ✨ Key Features

* 🖼️ Upload any web or mobile application screenshot

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/genesis-ai.git
```

```bash
cd genesis-ai
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn main:app --reload
```
